# IconButton

`IconButton`'s are used when you need an icon to trigger an action, such as showing a dialog or deleting an item.
It require a `label` that describes it's action. The `label` should be descriptive and succinct.
If you need an icon for purely decorative purposes use an `<Icon />` instead.

## Toggle

Icon buttons have the `toggle` prop which can be used to show a toggleable state. The `toggle` prop has to be explictly set to either `true` or `false`.

<Editor code={storyToggleOn} />
<Editor code={storyToggleOff} />

## ToggleBackground

Icon buttons have the `toggleBackground` prop which can be used to show a background color to the Icon. The `toggle` prop has to be explictly set to either `true` or `false`.

<Editor code={storyToggleBackground} />

### ToggleColor

Icon buttons have the `toggleColor` prop which can be used to specify what the icon's color and background color are when `toggle = true`. It accepts the following values:
`key` | `calculation` | `dimension` | `measure`
